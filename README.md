# Calorie tracker
This project consists of the code for a calorie tracker. Calculating how much a given person needs to eat in order to meet calorie goals based on their height, weight, activity level and Gender. It does NOT use BMI rather it uses TDEE which is a much more accurate way of tracking calories to ensure you are working towards your goal of either losing, gaining or maintaining weight.

### How to run the Programme:
There are two ways of running this programme. One way:

- Open any IDE (preferrably VScode)
- Clone the given web repository URL from with the use of Git: https://github.com/IesaK-tech/calorie-tracker.git

Second way:

- Download the files and open them in any IDE.

### What are the Files?

- `main.py` is the main file for the code where you will run the programme to see it work within the terminal
- `calorie_data.json` is the file which shows data persistence by saving the users input into the json file
- `gitlog.txt` is the file for my git history

There are no other Data files or anything else you need to download in order to run this programme.
